//-------------------------------
//BGM
//-------------------------------
プレイ中にBGMを流したい場合は以下の指示に従ってください。

１、【ネーム.exe】があるフォルダの中にMP3データを入れます。

２、【VisualStdio】などを使って【ソースファイル】内の【CP11_20151009_01_.sln】を開いてください。

３、【CP11_20151009_01_.sln】→【ソースファイル】内にある【main.cpp】を開いてください。

４、【main.cpp】内の78行目にある【/*】と122行目にある【*/】を消してください。

５、" "内の指示に従ってMP3データを拡張子まで記入してください。

６、終わり！閉廷！

※【error】が出る場合はMP3データをMP3にまた変換すると解決する場合があります。
